# おばけ面談室 v26

v25で出た「同じおばけ感」「文章の重複・破綻」「回答文の単調さ」を直すための濃縮版です。

## 変更点

- おばけ数を150体に抑制
- 15種類の人格パックを追加し、性格差を大きく振り分け
- 相談ケースを40種類に増量
- 回答文バリエーションを増加
- 同じ応答技法の連打減衰を維持
- 同じ未練でも、相談の題材・語り口・遺品が変わるよう調整
- `joinClean` で文章結合を校正し、句点や同じ語尾の重複を抑制
- 結果分析の相性傾向を「16タイプ名」から「人格パック名」寄りに変更

## 確認ポイント

- 同じおばけが続いている感じが薄くなったか
- 相談文や返答文に破綻・重複がないか
- 回答文の選択肢が前より違って見えるか
- 150体でも十分にバリエーションがあるか
