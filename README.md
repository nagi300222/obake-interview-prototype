# おばけ面談室 Prototype

スマホ縦画面向けのエンドレス面談ゲーム試作です。

## 遊び方

1. タイトル画面で「面談を始める」を押す
2. 10秒だけおばけの履歴書を見る
3. 面談開始後、30秒以内に4択から回答を選ぶ
4. 成仏回答なら成仏して次のおばけへ
5. 無難回答なら面談継続
6. 不信回答で不信が増え、地雷回答が増える
7. 地雷回答か時間切れでゲームオーバー

## 入っているもの

- 150体のおばけを自動生成
- 16タイプ風性格タグ
- タイトル画面
- 履歴書10秒表示
- 1回答30秒制限
- 成仏数カウント
- ポーズ機能
- 成仏演出
- ゲームオーバー時、相性のいい16タイプ風タグを表示

## 今回の調整

- 正解回答だけ長文でバレないよう、全回答を短めに統一
- 選択肢ボタンは2行以内に収まるよう調整
- 長文が枠をはみ出さないようにCSS側でも制御

## 注意

MBTI公式診断の再現ではなく、ゲーム内の「16タイプ風」性格タグとして実装しています。
公式設問・公式採点・公式レポートは使用していません。
