# おばけ面談室 Prototype

スマホ縦画面向けのエンドレス面談ゲーム試作です。

## 遊び方

1. タイトル画面から開始
2. 10秒だけ、おばけの履歴書を見る
3. 面談開始後、30秒以内に4択から返答を選ぶ
4. 成仏回答なら成仏して次のおばけへ
5. 無難な聞き方なら面談継続
6. 不信を招く返答で警戒が増え、地雷回答が増える
7. 地雷回答か時間切れでゲームオーバー

## 入っているもの

- 150体のおばけ生成
- 履歴書10秒表示
- 1回答30秒制限
- タイトル画面
- 成仏数カウント
- ポーズ機能
- 成仏演出
- ゲームオーバー時、相性のいい16タイプ風タグを表示

## v4 調整

- 文章を翻訳調から自然な短文会話へ調整
- 正解だけが詩的・抽象的に見えないよう修正
- 履歴書の性格タグ表示をやめ、入室前メモ/注意メモとして推理材料化
- 相談と回答を、診断文ではなく面談の聞き方に寄せた

## 注意

MBTI公式診断の再現ではなく、ゲーム内の「16タイプ風」性格タグとして実装しています。
公式設問・公式採点・公式レポートは使用していません。
