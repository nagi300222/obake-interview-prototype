# おばけ面談室 Prototype

スマホ縦画面向けのエンドレス面談ゲーム試作です。

## 遊び方

1. `index.html` をブラウザで開く
2. 10秒だけおばけの履歴書を見る
3. 面談開始後、30秒以内に4択から返答を選ぶ
4. おばけごとに内部で「成仏HP」と「不信HP」を持つ
5. 成仏HPが0になると成仏
6. 不信HPが0になる、地雷回答を選ぶ、時間切れになるとその面談は失敗
7. 成仏失敗が3回になるとゲームオーバー

## 回答タイプ

- 成仏回答：選ぶと一発成仏
- 求心回答：成仏HPを減らせる回答
- 様子見回答：基本的に両HPは減らない。長引くと不信が減ることもある
- 無難回答：おばけによっては不信HPが減る
- 不信回答：不信HPが減る
- 地雷回答：一発でその面談失敗

## 成仏の仕方

- 雑談成仏
- 叱責成仏
- 愚痴成仏
- 受け止め成仏
- 納得成仏
- 区切り成仏
- 承認成仏
- 思い出成仏
- 余白成仏
- 継承成仏

## 入っているもの

- 150体のおばけを自動生成
- 16タイプ風性格タグ
- 履歴書10秒表示
- 1回答30秒制限
- 成仏数カウント
- 成仏失敗3回までのライフ制
- ポーズ機能
- 成仏演出
- ゲームオーバー時、相性のいい16タイプ風タグと得意な成仏方法を表示

## 注意

MBTI公式診断の再現ではなく、ゲーム内の「16タイプ風」性格タグとして実装しています。
公式設問・公式採点・公式レポートは使用していません。
