# おばけ面談室 v25

v24を土台にした、具体相談＋性格差＋HP段階非表示＋ズレ反応強化版です。

## 追加・調整

- おばけごとに具体相談を追加
  - surface / detail / core / ask の段階式
- 「良いけど違う」反応を追加
- 汎用的に良い回答は成仏HPを大きく減らしすぎないよう調整
- 成仏タイプ一致時だけ大きく進むように調整
- pressure / pressureTolerance 追加
- 同じ応答技法の連打減衰を追加
- HP表示を段階化
  - 0〜4人成仏：数値表示
  - 5〜9人成仏：状態表示
  - 10人以降：HP非表示
- 履歴書の直接ヒントを少しぼかし、挙動ヒント寄りに変更
- 成仏時に具体アイテムの余韻を追加
- 結果分析にズレ回数とHP非表示メモを追加

## 確認ポイント

- 動作するか
- 相談が具体的になっているか
- 同じ回答だけでは成仏しにくくなっているか
- 良いけど違う反応が自然か
- 10人成仏後にHPが隠れるか
