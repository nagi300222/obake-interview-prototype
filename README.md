# おばけ面談室 Prototype v6

スマホ縦画面向けのエンドレス面談ゲーム試作です。

## 今回のv6更新

- 性別と名前の組み合わせを整理
  - 女性名 / 男性名 / 中性的な名前のプールを分離
- 年齢による性格の出方を追加
  - 十代、二十代、三十代、四十代、五十〜六十代、高齢で、履歴書メモや会話反応が少し変わります
- MBTI風16タイプ + BIG5風5因子を掛け合わせ
  - 16タイプ風：悩み方の骨格
  - BIG5風：話し方、反応、成仏しやすい聞き方のゆらぎ
- 同じ回答が同じおばけに何度も出にくいように修正
  - 面談中に一度表示された選択肢は、そのおばけでは基本的に再表示しません
  - 近いタグの回答も連続しにくくしています
- 履歴書の「答えっぽさ」を下げ、観察メモ寄りに調整

## 遊び方

1. `index.html` をブラウザで開く
2. 10秒だけおばけの履歴書を見る
3. 面談開始後、30秒以内に4択から回答を選ぶ
4. 成仏HPを削り、不信HPを守る
5. 成仏回答なら一発成仏
6. 地雷回答、時間切れ、不信HP切れでその面談は失敗
7. 3回成仏失敗でゲームオーバー

## 注意

MBTI公式診断やBIG5-BASIC公式診断の再現ではありません。
ゲーム内用の「16タイプ風」「BIG5風」性格パラメータとして独自実装しています。
公式設問、公式採点、公式レポートは使用していません。
