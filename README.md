# おばけ面談室 Prototype

スマホ縦画面向けのエンドレス面談ゲーム試作です。

## v3 変更点

- 正解だけ長文でバレる問題を修正
- 選択肢を「優しい/厳しい」ではなく、返答方針の違いに変更
- 相談文を直接的な悩みから、推理が必要な短い場面描写へ変更
- 履歴書から16タイプ風タグを隠し、遺品・周囲の証言・面談前の様子を推理材料に変更
- 地雷回答を露骨な暴言から「一見よさそうだが相手には刺さる言葉」に変更
- 履歴書10秒、回答30秒
- タイトル画面あり
- 選択肢は固定高さ＋2行制限で枠からはみ出しにくい設計

## 遊び方

1. `index.html` をブラウザで開く
2. タイトル画面で「面談を始める」
3. 10秒だけおばけの履歴書を見る
4. 面談開始後、30秒以内に4択から回答を選ぶ
5. 成仏回答なら成仏して次のおばけへ
6. 無難回答なら面談継続
7. ズレた回答で不信が増え、地雷回答が増える
8. 地雷回答か時間切れでゲームオーバー

## 注意

MBTI公式診断の再現ではなく、ゲーム内の「16タイプ風」性格タグとして実装しています。
公式設問・公式採点・公式レポートは使用していません。
